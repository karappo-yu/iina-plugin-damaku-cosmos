### 差异分析

| 问题 | 说明 |
|------|------|
| damaku-cosmos 行高是**比例值** | 如 1.2 表示 font-size 的 1.2 倍 |
| niconicomments 行高是**像素值** | 如 21px 是固定行高 |
| medium 模式差值最大 | damaku-cosmos: 1.16 (比例), niconicomments: 25.4px (像素) |
| Flash small 行高不同 | damaku-cosmos: 1.2, niconicomments: 1.2 (恰好相近) |

> **根本差异**: damaku-cosmos 用 CSS `line-height: 比例`，niconicomments 用 Canvas 固定像素值。当 font-size 变化时，两种方式渲染效果完全不同。
